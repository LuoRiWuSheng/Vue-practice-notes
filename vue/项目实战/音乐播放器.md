## 需求分析

- 推荐
    + 轮播图
    + 歌单列表
        * 歌单详情页
            - 顶部背景图
            - 歌曲列表
            - 随机播放按钮
            - 返回上一页面
- 歌手
    + 歌手列表
    + 右侧快速首字母入口-跳转
        * 歌手详情页（和歌单详情页一样）
- 排行
    + 歌曲榜单（榜单分类）列表
        * 榜单详情（一类歌曲的列表）（个歌单详情页一样）
            - 多一个排序样式（1-3名显示icon，4+ 显示数字）
- 搜索
    + 历史搜索列表
    + 热门搜索

## vue上手

### vue-cli脚手架安装

我的电脑是vue-cli 3.X版本的     
拉取2.X的模版需要安装一个中间插件

**脚手架切换**

```js
    npm install -g @vue/cli-init
    # `vue init` 的运行效果将会跟 `vue-cli@2.x` 相同
    vue init webpack my-project
```
